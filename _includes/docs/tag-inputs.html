Tag Input

constructor

{% highlight js %} const tagInput = new TagInput(dom, options) {% endhighlight %}
名稱 型別 說明 預設值
dom HTMLElement 需要設定成 TagInput 的元素
options object 選項 {}
options.validate function | async function 驗證輸入的字串是否可以變為標籤，回傳物件 { isTag: Boolean }，如果有指定標籤的 classname 可以回傳 { isTag: Boolean, classname: String } () => ({ isTag: true })
options.change function 標籤有異動時會呼叫這個 callback，回傳標籤陣列 [{ elem, remove, res }...]

destroy

移除 TagInput 產生的 dom 元素與 event listeners

{% highlight js %} tagInput.destroy() {% endhighlight %}
