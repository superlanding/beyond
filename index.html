Beyond

這裡放全站共用元件

Ruby / Rails

使用方式

{% highlight ruby %} gem "beyond-rails" {% endhighlight %}
在 application.js
{% highlight js %} //= require beyond {% endhighlight %}
在 application.scss
{% highlight scss %} @import "beyond-sprockets"; @import "beyond"; {% endhighlight %}

Webpack

安裝

{% highlight shell %} yarn add @superlanding/beyond --exact {% endhighlight %}

使用

在 sass 檔案引入樣式
{% highlight scss %} // 定義 css 圖片路徑與 font icons 路徑 $beyond-icon-font-path: '~@superlanding/beyond/src/font/'; $beyond-css-img-path: '~@superlanding/beyond/src/sass/img/'; @import '~@superlanding/beyond/src/sass/index.scss'; {% endhighlight %}

JS 元件存取

自動載入

  • 必須在 document ready 之後執行
  • 自動載入目前只支援以下元件:
    • Alert
    • Checkbox
    • Dropdown
    • Menu
    • Navbar
    • Radio
    • Sidebar
    • Tabbox
    • Tooltip
JavaScript Modules
{% highlight js %} import { bind, unbindAll } from '@superlanding/beyond' // bind 執行後會返回一個 unbind 方法 // unbind 只會 destroy 透過 bind 綁定的元件 const unbind = bind() // 或是可以使用 unbindAll 解除所有載入 Beyond 後所生成的元件 unbindAll() {% endhighlight %}
Browser
{% highlight js %} const unbind = window.beyond.bind() window.beyond.unbindAll() {% endhighlight %}

手動載入

JavaScript Modules
{% highlight js %} import { Datepicker, Dropdown, Modal, Tabbox, Toast, Tooltip } from '@superlanding/beyond' {% endhighlight %}
Browser
{% highlight js %} const { Datepicker, Dropdown, Modal, Tabbox, Toast, Tooltip } = window.beyond {% endhighlight %}
