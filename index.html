Beyond

這裡放全站共用元件

Ruby / Rails

使用方式

{% highlight ruby %} gem "beyond-rails" {% endhighlight %}
在 application.js
{% highlight js %} //= require beyond {% endhighlight %}
在 application.scss
{% highlight scss %} @import "beyond-sprockets" @import "beyond" {% endhighlight %}

Webpack

安裝

{% highlight shell %} yarn add @superlanding/beyond --exact {% endhighlight %}

使用

在 sass 檔案引入樣式
{% highlight scss %} // 定義 css 圖片路徑與 font icons 路徑 $beyond-icon-font-path: '~@superlanding/beyond/src/font/'; $beyond-css-img-path: '~@superlanding/beyond/src/sass/img/'; @import '~@superlanding/beyond/src/sass/index.scss'; {% endhighlight %}
JS 元件存取
{% highlight js %} import { Datepicker, Dropdown, Modal, Tabbox, Toast, Tooltip } from '@superlanding/beyond' // 或是在瀏覽器環境存取 window.beyond const { Datepicker, Dropdown, Modal, Tabbox, Toast, Tooltip } = window.beyond {% endhighlight %}
