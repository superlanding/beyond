# Beyond
https://superlanding.github.io/beyond/

## Site Development

```bash
bundle
yarn
foreman start
# http://localhost:4000/
```

## Site Deployment

```bash
yarn deploy
```

## Site Release

```bash
# Do this in master branch.
# It will build assets,
# publish npm version and deploy to github page.
yarn release
```
